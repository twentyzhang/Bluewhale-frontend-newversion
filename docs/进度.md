## 建议实现顺序

```
1. JWT 基础设施（依赖 → 工具类 → Filter → SecurityConfig 接入）
2. 用户模块（注册 + 登录，产出可用 Token，其余模块都需要鉴权）
3. 商品浏览（分类树、商店列表、商品列表/详情/搜索，无需登录，便于早期联调）
4. 购物车模块
5. 收货地址模块
6. 商店管理 + 商品管理（Staff/Admin 操作）
7. 订单模块（依赖购物车、地址、商品库存）
8. 优惠券模块（领取 + 下单时核销，依赖订单模块）
9. 评论模块（依赖订单完成状态）
10. 报表模块（依赖订单数据）
```
